## Развертывание собственного экземпляра Fresco

**1. Настройка необходимых служб**

- Создайте базу данных с [PlanetScale](https://planetscale.com/docs/tutorials/planetscale-quick-start-guide)
- Создайте новое приложение на [uploadthing](https://uploadthing.com/)

**2. Используйте кнопку развертывания Vercel для настройки развертывания вашего проекта в Vercel.**

[![Развернуть с Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fcomplexdatacollective%2FFresco%2Ftree%2Ffeature%2Finitial-setup-flow&env=DATABASE_URL,UPLOADTHING_SECRET,UPLOADTHING_APP_ID)

**Примечание: Для безопасности у вас есть _пять минут_ с момента развертывания приложения для создания учетной записи пользователя. Если этот срок истекает без создания учетной записи пользователя, ваша конфигурация устареет. Вы можете повторно развернуть, следуя тем же шагам.**

**3. Создание репозитория Git**

Следуйте инструкциям по созданию репозитория Git для развертывания. В нем будет клонированная версия этого репозитория.

**4. Настройка проекта**

Укажите необходимые переменные среды из служб, которые вы настроили на шаге 1.

| Переменная         | Описание                                                                                        |
| ------------------ | ----------------------------------------------------------------------------------------------- |
| DATABASE_URL       | [Строка подключения к базе данных](https://planetscale.com/docs/concepts/connection-strings)    |
| UPLOADTHING_SECRET | Ключ API для вашего [приложения uploadthing](https://uploadthing.com/dashboard)                 |
| UPLOADTHING_APP_ID | Идентификатор приложения для вашего [приложения uploadthing](https://uploadthing.com/dashboard) |

**5. Развертывание**

Нажмите "Развернуть" и дождитесь завершения развертывания.

**6. Создание учетной записи пользователя**

Посетите ваше развернутое приложение, чтобы создать учетную запись вашего администратора. Может быть создана только одна учетная запись пользователя.
